# Grab Fare Calculator

This is an interactive C++ program for the LDCW6123 Group Project.

## Purpose
The program estimates a Grab ride fare based on:
- Pickup location
- Drop-off location
- Pickup time
- Grab service type
- Promo code

## Features
- Location selection
- Automatic distance and duration
- Peak hour detection
- Service type selection
- Promo code discount
- Final receipt display
